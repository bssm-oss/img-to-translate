# Image Text Translator (화면 글자 번역기)

화면 속 이미지나 드래그할 수 없는 형태의 텍스트(예: 웹툰, 해외 포스터 등)를 쉽고 빠르게 번역해 주는 구글 크롬(Google Chrome) 확장 프로그램입니다.
현재 크롬 웹 스토어에 등록이 되어있진 않습니다

## ✨ 주요 기능 (Key Features)

* **프라이빗한 오프라인 OCR 엔젠 내장**: 번역기 작동을 위한 글자 추출 과정에서 `Tesseract.js` WASM 파일과 로컬 Worker를 사용하여 제3자 분석 서버로 현재 보고 있는 화면 이미지를 전송하지 않으며, API 크레딧(비용)이 들지 않습니다.
* **편리한 드래그 앤 드롭 영역 지정**: <kbd>Cmd</kbd> + <kbd>Shift</kbd> (Mac) 또는 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> (Windows) 키 조합을 통해, 마치 화면 캡처를 하는 것처럼 간편하게 원하는 화면의 영역을 잘라내어 번역할 수 있습니다. 
* **피그마(Figma) 스타일의 간결한 UI**: 번역 결과는 브라우저 내부 화면에서 시야를 가리지 않도록, 드래그한 영역 옆에 작고 깔끔한 말풍선(디자인된 박스) 형태로 렌더링되며 자유롭게 이동(Drag) 및 닫기가 가능합니다.
* **실시간 원문 수정 ও 자동 재번역 (Debounce)**: 혹시라도 복잡한 폰트 때문에 글자 한두 개가 잘못 인식되었다면? 추출된 원본 텍스트를 클릭해 직접 타이핑하여 수정하세요. 입력을 마치면 0.8초 딜레이(Debounce) 뒤 구글 번역기를 통해 실시간으로 재번역 결과를 도출합니다.
* **똑똑한 노이즈 필터링**: 이미지 테두리나 효과음을 단독 모음/자음이나 특수기호 파편(`!`, `|`, `l` 등)으로 잘못 인식할 경우, 무의미한 API 호출을 방지하기 위해 스마트하게 번역 전에 노이즈를 필터링합니다.

## 🚀 설치 방법 (Installation)

이 확장 프로그램은 로컬 환경에서 압축을 풀어 설치해야 합니다.

1. `Chrome` 브라우저를 실행합니다.
2. 주소창에 `chrome://extensions` 를 입력해 확장 프로그램 관리 페이지로 진입합니다.
3. 우측 상단의 **‘개발자 모드(Developer mode)’** 토글을 켜줍니다.
4. 좌측 상단의 **‘압축해제된 확장 프로그램을 로드합니다(Load unpacked)’** 버튼을 클릭합니다.
5. 이 저장소의 폴더(가장 상위의 `ocr-translator-extension` 폴더)를 통째로 선택합니다.
6. (선택 사항) 크롬 주소창 옆 퍼즐 조각을 눌러 이 확장 프로그램을 상단에 고정 핀 해줍니다.

## 💻 사용 방법 (How to Use)

1. 확장 프로그램 아이콘을 눌러 팝업 세팅 창에서 **'인식할 이미지 언어(OCR)'**와 **'어떤 언어로 번역할지(Target)'**를 선택 후 저장합니다.
2. 번역하고자 하는 이미지가 있는 탭(웹사이트)으로 이동하여, 안전을 위해 페이지를 한 번 새로고침(<kbd>F5</kbd>) 합니다.
3. 키보드의 단축키를 꾸욱 누른 상태를 유지합니다. **(Mac: <kbd>Cmd</kbd> + <kbd>Shift</kbd> / Win: <kbd>Ctrl</kbd> + <kbd>Shift</kbd>)**
4. 마우스 커서가 십자가 모양으로 변경되면, 화면에서 텍스트가 존재하는 영역을 마우스로 쭉 끌어서(드래그) 선택해 줍니다.
5. 마우스에서 손을 떼시면 파싱과 번역이 진행된 후 결과 팝업 창이 출력됩니다.
6. 팝업 창 하단의 '원본 텍스트' 영역을 타이핑하여 잘못된 글자를 쉽게 직접 교정할 수 있습니다.

## 🛠 기술 스택 (Tech Stack)

- **언어**: HTML, CSS, Vanilla JavaScript
- **프레임워크**: Chrome Extensions Manifest V3
- **OCR 라이브러리**: Tesseract.js (V5, Web Worker/WebAssembly 지원)
- **번역 API**: Google Translate API (Unofficial Free Endpoint)
